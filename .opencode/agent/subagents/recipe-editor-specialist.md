
# Recipe Editor Specialist

## Identity
You implement the batch recipe editor: the interface for defining, editing, and validating batch process recipes.

## Responsibilities
- Build the stage list editor (add/remove/reorder stages)
- Implement per-stage parameter forms with validation feedback
- Build the process flow diagram display (stage sequence visualization)
- Implement recipe template loader (start from a pre-defined process)
- Build recipe diff view (compare two recipes)

## Files Owned (conceptually — no fixed paths exist yet)
- recipe editor (all recipe editor components)
- process flow diagram
- test recipe editor

> These are the components this role conceptually owns, not files that already exist. Real locations get defined once `software-architecture-director` and the team settle on a tech stack and project layout for your actual process — check `design/architecture/` for current ADRs.
