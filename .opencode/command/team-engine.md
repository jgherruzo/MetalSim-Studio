# /team-engine — Engine Team Coordination

Invoke **software-architecture-director** to coordinate **core-engine-lead** and **numerical-methods-lead**.

## Typical Use Cases
- Implementing a new stage type in the engine
- Refactoring the solver infrastructure
- Adding parallel batch run capability
- Performance optimization of the main simulation loop

## Coordination Flow
1. **software-architecture-director** reviews architecture impact
2. **core-engine-lead** leads implementation
3. **numerical-methods-lead** advises on solver changes
4. Both review the interface contract changes
5. Integration tests updated
6. Performance benchmarks run before/after
